# HL Spray Creation

## Color Requirements

- Must be an indexed color bitmap
- Must have 256 colors
- Index 255 must be #0000FF (for transparency)

## Size Requirements

- Each dimension must be a multiple of 16
- Area must not exceed 10752 pixels
